# WorldDev-order-service
Cloud Native Application for managing orders of worlddev's bookshop system

## _Functionalities_

* Order a book
* Fetch all orders

## _Tech Stack_

* Java 17
* Spring Framework 6
* SpringBoot 3
* Sping Cloud
* Reactive Spring
* Docker
* Kubernetes
* PostgreSQL
* FlyWay
* Gradle
* Tilt
* Github Actions
